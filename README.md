# ValentineChocolates

A fun and interactive Valentine-themed chocolate box game built with L√ñVE (Love2D) and Lua.

## Features
- Heart-shaped chocolate slots
- Customizable chocolates
- Valentine-themed assets

## Getting Started

### Prerequisites
- [L√ñVE (Love2D)](https://love2d.org/) installed on your system

### Running the Game
1. Download or clone this repository.
2. Open a terminal and navigate to the project directory.
3. Run the game with:
   ```
   love .
   ```

## File Structure
- `main.lua` - Entry point of the game
- `conf.lua` - L√ñVE configuration
- `chocolate.lua` - Chocolate logic and rendering
- `heartSlot.lua` - Heart slot logic
- `utils.lua` - Utility functions
- `assets/` - Game assets (images, sounds, etc.)

## Customization
- Add your own chocolate designs or assets in the `assets/` folder.
- Modify `chocolate.lua` and `heartSlot.lua` to change game logic or visuals.
